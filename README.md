# Описание
Приложение для поиска информации о пользователе на GitHub.<br>
Используется GitHub API.<br>
Реализовано с помощью React + Redux.

# Запуск
Для запуска приложения используйте поочередно команды:
1. <b>npm install</b>
1. <b>npm start</b>
